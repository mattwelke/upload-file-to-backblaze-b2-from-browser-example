B2 Image Browser Upload Example

Choose a file to upload and click "Upload".

Observe in the network tab as the following steps are performed:

  1. The upload details are fetched from the running back end.
  2. The preflight (OPTIONS) request for the B2 file upload is performed.
  3. The POST request for the B2 file upload is performed to the upload URL returned from the running back end.

Upload file result:

Upload succeeded (2xx response from b2_upload_file API)
